# Operating Systems Used
This project was developed on Linux (Fedora distribution) and Windows 11.

# Technology Stack
The project is written in Rust and was initialized with Cargo, Rust's package manager and build tool.

# How to Run This Project
It is very simple: make sure you are inside the project folder and run:

`cargo run`

# Brief Explanation
This program reads a string from the terminal and checks whether it is accepted by the lexical analyzer automaton using the alphabet `{a, b}`.
